# Data model

## Task object

A `task` is represented by the following JSON object:

```json
{
  "id": int,
  "user_id": int,
  "title": string,
  "state": string,
  "due_at": datetime,
  "created_at": datetime,
  "updated_at": datetime,
  "position": int,
  "overdue": boolean
}
```
